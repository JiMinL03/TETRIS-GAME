# Tetris

## MFC를 이용한 테트리스 게임

## 1. 개발 환경

- OS : Windows 10
- IDE : Visual Studio 2019
- Language : C++

## 2. 조작법

- 방향키 ← → : 좌우 이동
- 방향키 ↑ : 회전
- 방향키 ↓ : 빠른 낙하
